This directory contains an example batch run that demonstrates the
--infertransfers and --phylogenomics options.

#####################
FILES:

GeneTrees
  Directory: Twenty-three gene trees sampled from 11 cyanobacterial species.

README.txt
  This file.

batch-binary.txt
  Specification file for a batch analysis with a binary species tree.

batch-nonBinary.txt
  Specification file for a batch analysis with a non-binary species tree.

speciestree_Cyano.nwk
  Species tree for 11 cyanobacterial species.

speciestree_Cyano-nonbinary.nwk
  Species tree for 11 cyanobacterial species with one non-binary node.

sample_commands.txt
  Example commands for various analyses using running Notung from the
  command line with a batch file.

SOURCE:

    al. 2006. Phylogenetic analyses of cyanobacterial genomes:
    quantification of horizontal gene transfer events. Genome Res. 16,
    1099-1108.
